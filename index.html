PocketChef was created to adress the inconvenience of online recipe searching and grocery list creation. The aim is to allow users to search recipes and convert the recipe to an editable shopping list that can be saved for later use. The app was built in Rails, ruby, javascript, and utilized the Edamam API. This is still an ongoing project. To see what features will be added please visit my gitHub :)

Nanny Dunkelheit is an interactive storytelling experience based on a game I am working on. This was created in Sinatra with vanilla javascript and pure css animations. Sendgrid API was incorporated for emails and Facebook plug-ins to connect with social media.Images created in photoshop and music sourced was edited in Audacity.
